Project Collaborators

Roles

Each project can have many collaborators. Collaborators can view the project, and edit the metadata and files.

One of these collaborators is the owner. The owner may:

  • Submit the project for publication
  • Delete the project
  • Invite and remove collaborators
  • Assign ownership of the project to another collaborator
